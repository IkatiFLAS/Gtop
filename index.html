Monitor de Sistema

GTOP es una herramienta CLI elegante y potente para monitorear recursos del sistema en tiempo real. Información de CPU, RAM, disco, red y procesos, todo en una interfaz hermosa.

Características

Todo lo que necesitas para monitorear tu sistema

⚡

CPU en Tiempo Real

Monitorea el uso de CPU con actualización en tiempo real y visualización de barras de progreso.

🧠

Uso de Memoria

Visualiza el consumo de RAM con porcentaje detallado y capacidad total del sistema.

💾

Espacio en Disco

Controla el uso de tus discos con información precisa del espacio usado y disponible.

🌡️

Temperatura

Visualiza la temperatura del sistema y recibe alertas cuando los valores son críticos.

🌐

Estadísticas de Red

Monitorea el tráfico de red entrante y saliente en KB/s con actualización continua.

📋

Gestión de Procesos

Navega, busca y termina procesos directamente desde la interfaz con confirmación de seguridad.
